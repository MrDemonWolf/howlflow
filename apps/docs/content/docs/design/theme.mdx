# Theme

HowlFlow uses a dark-first design with cyan and gold accents.

## Colors

| Token | Hex | Usage |
|-------|-----|-------|
| `bg-primary` | `#0A0E1A` | Main background |
| `bg-secondary` | `#111827` | Cards, surfaces |
| `bg-tertiary` | `#1F2937` | Elevated surfaces |
| `accent-cyan` | `#00D4FF` | Primary actions |
| `accent-gold` | `#FFB800` | Achievements, wolf |
| `accent-green` | `#10B981` | Success states |
| `accent-red` | `#EF4444` | Destructive actions |
| `text-primary` | `#F9FAFB` | Body text |
| `text-secondary` | `#9CA3AF` | Secondary text |

## Typography

The primary font is **Outfit** (variable weight). Used across all text in the app.

## Spacing Scale

4 / 8 / 16 / 24 / 32 / 48 px mapped to xs / sm / md / lg / xl / 2xl tokens.
