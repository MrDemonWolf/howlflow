# Anti-Shame Principles

HowlFlow is designed specifically for people with ADHD. Every feature must pass this test:

> "Would this make someone feel bad about themselves if they stopped using the app for a week?"

If yes, redesign it.

## Guidelines

1. **No punitive streaks** — Track consistency without punishment. Missing a day should never feel like losing progress.

2. **Gentle notifications** — Notifications suggest, never demand. "Your wolf misses you" not "You broke your streak!"

3. **No comparisons** — No leaderboards, no social features that create pressure.

4. **Celebrate small wins** — Completing one task is worth celebrating. The wolf is always happy to see you.

5. **Easy restart** — Coming back after a break should feel welcoming, not overwhelming. No "catch up" mechanics.

6. **Honest time tracking** — Show actual focus time without judgment. 5 minutes of focus is still focus.
